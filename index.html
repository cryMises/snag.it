How it works
1
Scrape Bright Data pulls the 5 most recent posts from Instagram every day at noon, filtered to the last 7 days.
2
Extract Mistral OCR reads text from post images. Gemini classifies each post and extracts event title, date, location, and registration info.
3
Serve Confirmed events land in a clean table, served via webhook API and displayed here as swipeable cards.
